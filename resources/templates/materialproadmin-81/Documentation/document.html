Introduction

MaterialPro Bootstrap Dashboard is built with Bootstrap-5.
We have included PSD file which can help our buyers for the extension of their project development.
This theme is fully operable and handy if the buyer and their team has basic knowledge of JS, CSS, HTML.
The theme can be customized as per buyer's requisites.
If the buyer has an inquiry while going through the documentation, they can create a support ticket from wrappixel.com. Our Tech team will revert back at the earliest.

Support

We at WrapPixel offer affordable Customization Services for our templates. For any customization work, please email us at info@wrappixel.com.
Kindly make sure that the support ticket that has been created is regarding support only and not any customization.
The Buyer can refer the entire documentation and check out whether the bug/query that has been posted is not mentioned in our documentation.
